Tips for recording system/driver output:

If you are using an external sound source (Mixing console) make sure the default system input
is selected in sound settings.
Your brower will only let you record the default audio input for your system

Apps for re-routing sound sources from default to something interesting:
Linux: Pulse audio
Mac: Soundflower
Win: Cable
